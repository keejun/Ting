Ting

基于音乐api的音乐搜索应用

View the Project on GitHub keejun/Ting

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

  • 此应用基于豆瓣网（douban.com），虾米网（xiami.com）和其他网站的站内api开发.获取外站的数据格式为json.

  • 通过ajax获取调用资源并使用jquery进行遍历获取。服务器端采用为Java和oracle database .用于存取用户喜欢的音乐。 并提供音乐话题交流，收听音乐和在浏览器上直接播放硬盘内音乐的服务。

  • 部分模块采用了html5进行了开发，调用的基于jquery的js组件比较多。在谷歌和火狐浏览器下调试通过。

  • 此应用采用的语言：Javascript、Java ，属于二次开发的音乐应用
